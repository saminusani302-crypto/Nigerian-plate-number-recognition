PROJECT OVERVIEW
================

This is a production-ready, AI-powered system for detecting, localizing, and 
recognizing Nigerian license plates in real-time video feeds and static images.

The system combines state-of-the-art deep learning models (YOLOv8) with advanced
OCR technology (EasyOCR) to provide accurate, fast license plate recognition with
comprehensive logging and web-based interface.

NIGERIAN PLATE FORMAT
====================

Standard Format: ABC 123 XY
  - ABC:   State/Region code (2-3 letters)
  - 123:   Sequential number (3 digits)
  - XY:    Type code (2 letters)
            AA-AZ: Private vehicles
            BA-BZ: Government vehicles
            CA-CZ: Commercial vehicles

The system validates detected plates against this format.

MODEL VARIANTS
==============

YOLOv8 Models Supported:
  - yolov8n.pt   (Nano:    5.3M params,  fastest, least accurate)
  - yolov8s.pt   (Small:   11.2M params, balanced)
  - yolov8m.pt   (Medium:  25.9M params, slower, most accurate)
  - yolov8l.pt   (Large:   43.7M params, very slow)
  - yolov8x.pt   (Extra:   68.2M params, slowest, best accuracy)

Recommendation: Use yolov8n.pt for real-time, yolov8m.pt for accuracy.

LOGGING FORMAT
==============

CSV Columns:
  id                    - Unique entry ID
  timestamp             - ISO format timestamp
  plate_number          - Raw OCR output
  formatted_plate       - Formatted plate number
  confidence            - OCR confidence (0-1)
  is_valid              - Validation result (true/false)
  frame_source          - Source (video, webcam, image)
  frame_index           - Frame number in sequence
  detection_confidence  - YOLOv8 confidence (0-1)
  notes                 - Additional notes

JSON Structure:
  {
    "id": "20240115120530123456",
    "timestamp": "2024-01-15T12:05:30.123456",
    "plate_number": "ABC123XY",
    "formatted_plate": "ABC 123 XY",
    "confidence": 0.92,
    "is_valid": true,
    ...
  }
